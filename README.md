# lita-bitbucket-wehbhook

A Lita handler to send Bitbucket.org commit messages to the chat.

## Installation

Add lita-bitbucket-wehbhook to your Lita instance's Gemfile:

``` ruby
gem "lita-bitbucket-wehbhook"
```
